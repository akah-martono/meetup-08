# Update Software

## Update Software Package
Memperbarui daftar paket software pada linux
```bash
sudo apt update -y
```
Download dan install paket terbaru
```bash
sudo apt dist-upgrade -y
```
Menghapus paket-paket software yang sudah tidak terpakai
```bash
sudo apt autoremove
```

## Reboot Server
```bash
sudo reboot now
```
